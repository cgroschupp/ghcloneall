Script to clone/update all ZopeFoundation repos from GitHub.

Usage::

    mkdir ~/src/zf
    cd ~/src/zf
    wget https://github.com/mgedmin/cloneall/blob/master/clone_all_zf_repos.py -O cloneall.py
    ./cloneall.py

Example output::

    + AccessControl
    + Acquisition
    + AuthEncoding
    + BTrees
    + CMF-buildout
    + DateTime
    + DocumentTemplate
    + ExtensionClass
    + Missing
    [                    ] 9/339
